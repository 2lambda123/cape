1r""" 

2:mod:`cape.pyover.options.util`: Utilities for pyOver options module 

3===================================================================== 

4 

5This module provides tools to read, access, modify, and write settings for 

6:mod:`cape.pyover`. It is based off of the :mod:`cape.cfdx.options.util` module and 

7provides a special class :class:`cape.cfdx.options.odict` that is subclassed from 

8the Python built-in :class:`dict`. Behavior, such as ``opts['OverNamelist']`` 

9or ``opts.get('OverNamelist')`` are also present. In addition, many convenience 

10methods such as ``opts.get_OverNamelist()`` are provided. 

11 

12In addition, this module controls default values of each pyOver parameter in a 

13three-step process. The precedence used to determine what the value of a given 

14parameter should be is below. 

15 

16 #. Values directly specified in the input file, :file:`pyOver.json` 

17  

18 #. Values specified in the default control file, 

19 :file:`$PYOVER/settings/pyOver.default.json` 

20  

21 #. Hard-coded defaults from this module 

22  

23:See Also: 

24 * :mod:`cape.cfdx.options.util` 

25 * :mod:`cape.pyover.options` 

26"""
