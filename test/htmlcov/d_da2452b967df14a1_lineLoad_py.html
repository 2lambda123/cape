1""" 

2:mod:`cape.pycart.lineLoad`: Sectional loads module 

3==================================================== 

4 

5This module contains functions for reading and processing sectional loads. It 

6is a version of :mod:`cape.cfdx.lineLoad` that is closely tied to 

7:mod:`cape.pycart.dataBook`. 

8 

9It provides the primary class :class:`DBLineLoad`, which 

10is a subclass of :class:`cape.cfdx.dataBook.DBBase`. This class is an interface to 

11all line load data for a specific surface component. 

12 

13For reading the sectional load for a single solution on one component (which 

14includes multiple coefficients), there is another class :class:`CaseLL`. 

15 

16Finally, reading seam curves from individual cases utilizes the class 

17:class:`CaseSeam`. 

18 

19:See also: 

20 * :mod:`cape.cfdx.lineLoad` 

21 * :mod:`cape.cfdx.dataBook` 

22 * :mod:`cape.pycart.dataBook` 

23"""
