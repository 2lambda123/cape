2:mod:`cape.filecntl.namelist2`: Fortran namelists with repeat sections 

3======================================================================= 

4 

5This is a module built off of the :mod:`cape.filecntl.FileCntl` module 

6customized for manipulating Fortran namelists. Such files are split 

7into sections which are called "name lists." Each name list has syntax 

8similar to the following. 

9 

10 .. code-block:: none 

11 

12 $FLOINP 

13 FSMACH = 4.0, 

14 ALPHA = 1.0, 

15 BETA = 0.0, 

16 $END 

17 

18and this module is designed to recognize such sections. The main 

19feature of this module is methods to set specific properties of a 

20namelist file, for example the Mach number or CFL number. 

21 

22The difference between this module and 

23:class:`cape.filecntl.namelist.Namelist` is that this module can support 

24multiple namelists with the same title. This is particularly important 

25for Overflow, which has ``GRDNAM``, ``BCINP``, and other sections 

26defined for each structured grid. These modules should be combined as 

27the differing namelist syntaxes are actually part of one file 

28 convention. 

29 

30This function provides a class 

31:class:`cape.filecntl.namelist2.Namelist2` that can both read and set 

32 values in the namelist. The key functions are 

33 

34 * :func:`Namelist2.GetKeyFromGroupName` 

35 * :func:`Namelist2.GetKeyFromGroupIndex` 

36 * :func:`Namelist2.SetKeyInGroupName` 

37 * :func:`Namelist2.SetKeyInGroupIndex` 

38 

39The conversion from namelist text to Python is handled by 

40:func:`Namelist2.ConvertToText`, and the reverse is handled by 

41:func:`Namelist2.ConvertToVal`. Conversions cannot quite be performed 

42just by the Python functions :func:`print` and :func:`eval` because 

43delimiters are not used in the same fashion. Some of the conversions are 

44tabulated below. 

45 

46 +----------------------+------------------------+ 

47 | Namelist | Python | 

48 +======================+========================+ 

49 | ``val = "text"`` | ``val = "text"`` | 

50 +----------------------+------------------------+ 

51 | ``val = 'text'`` | ``val = 'text'`` | 

52 +----------------------+------------------------+ 

53 | ``val = 3`` | ``val = 3`` | 

54 +----------------------+------------------------+ 

55 | ``val = 3.1`` | ``val = 3.1`` | 

56 +----------------------+------------------------+ 

57 | ``val = .false.`` | ``val = False`` | 

58 +----------------------+------------------------+ 

59 | ``val = .true.`` | ``val = True`` | 

60 +----------------------+------------------------+ 

61 | ``val = .f.`` | ``val = False`` | 

62 +----------------------+------------------------+ 

63 | ``val = .t.`` | ``val = True`` | 

64 +----------------------+------------------------+ 

65 | ``val = 10.0 20.0`` | ``val = [10.0, 20.0]`` | 

66 +----------------------+------------------------+ 

67 | ``val = 1, 100`` | ``val = [1, 100]`` | 

68 +----------------------+------------------------+ 

69 | ``val(1) = 1.2`` | ``val = [1.2, 1.5]`` | 

70 +----------------------+------------------------+ 

71 | ``val(2) = 1.5`` | | 

72 +----------------------+------------------------+ 

73 | ``val = _mach_`` | ``val = "_mach_"`` | 

74 +----------------------+------------------------+ 

75 

76In most cases, the :class:`Namelist` will try to interpret invalid values for 

77any namelist entry as a string with missing quotes. The reason for this is 

78that users often create template namelist with entries like ``_mach_`` that can 

79be safely replaced with appropriate values using ``sed`` commands or something 

80similar. 

81 

82See also: 

83

139 # Function to update the namelists 

140 def UpdateNamelist(self): 

141 r"""Update the line indices for each namelist 

142 

143 :Call: 

144 >>> nml.UpdateNamelist() 

145 :Inputs: 

146 *nml*: :class:`cape.filecntl.namelist2.Namelist2` 

147 Interface to namelist with repeated lists 

148 :Versions: 

149 * 2016-01-29 ``@ddalle``: Version 1.0 

150 """ 

151 # Find the lines that start the lists 

152 I0 = np.array(self.GetIndexSearch(r'\s*[&$]'), dtype=int) 

153 # Find the lines that end with '/' 

154 J0 = np.array(self.GetIndexSearch(r'.*/\s*$'), dtype=int) 

155 # Of those, find those that are comments 

156 J1 = np.array(self.GetIndexSearch(r'\s*!.*/\s*$'), dtype=int) 

157 # These are the namelist end lines using '/' 

158 J = np.setdiff1d(J0, J1) 

159 # Get start and end keywords to each line 

160 grpnm = np.array([self.lines[i].split()[0][1:] for i in I0]) 

161 kwbeg = np.array([self.lines[i].split()[0][1:].lower() for i in I0]) 

162 kwend = np.array([self.lines[i].split()[-1][1:].lower() for i in I0]) 

163 # Save the start indices 

164 self.ibeg = I0[kwbeg != "end"] 

165 # Save the end indices 

166 self.iend = np.sort(np.hstack((I0[kwend == "end"], J))) 

167 # Save the names 

168 self.Groups = grpnm[kwbeg != "end"]

170 # Apply a whole bunch of options 

171 def ApplyDict(self, opts): 

172 r"""Apply a whole dictionary of settings to the namelist 

173 

174 :Call: 

175 >>> nml.ApplyDict(opts) 

176 :Inputs: 

177 *nml*: :class:`cape.filecntl.namelist2.Namelist2` 

178 Old-style namelist inerface 

179 *opts*: :class:`dict` 

180 Dictionary of namelist options 

181 :Versions: 

182 * 2016-02-01 ``@ddalle``: Version 1.0 

183 """ 

184 # Loop through major keys. 

185 for grp in opts.keys(): 

186 # Check for illegal name to specify which index to use 

187 if "&" in grp: 

188 try: 

189 # Split into two parts 

190 ggrp, igrp = grp.split("&") 

191 # Convert to integer 

192 igrp = int(igrp)-1 

193 except Exception: 

194 # Some sort of error 

195 raise ValueError("Illegal namelist group '%s'" % grp) 

196 else: 

197 # Assume first grip 

198 ggrp = grp 

199 igrp = 0 

200 # Check it it's an existing group 

201 if ggrp not in self.Groups: 

202 # Initialize the section. 

203 self.InsertGroup(0, ggrp) 

204 # Loop through the keys in this subnamelist/section 

205 for k in opts[grp].keys(): 

206 # Set the value. 

207 self.SetKeyInGroupName(ggrp, k, opts[grp][k], igrp=igrp)

210 def InsertGroup(self, igrp, grp): 

211 r"""Insert a group as group number *igrp* 

212 

213 :Call: 

214 >>> nml.InsertGroup(igrp, grp) 

215 :Inputs: 

216 *nml*: :class:`cape.filecntl.namelist2.Namelist2` 

217 Interface to namelist with repeated lists 

218 *igrp*: :class:`int` 

219 Index of location at which to insert group 

220 *grp*: :class:`str` 

221 Name of the group to insert 

222 :Versions: 

223 * 2016-02-01 ``@ddalle``: Version 1.0 

224 """ 

225 # Test for "append" 

226 if igrp >= len(self.ibeg) or igrp == -1: 

227 # Making a new group 

228 ibeg = self.iend[-1]+1 

229 iend = self.iend[-1] 

230 else: 

231 # Get the index of the group that we insert before 

232 ibeg = self.ibeg[igrp] 

233 iend = self.iend[igrp] 

234 # Query the current starting character 

235 gchar = self.lines[ibeg].lstrip()[0] 

236 # Check end line type 

237 if self.lines[iend].rstrip().endswith('/'): 

238 # Ends with a '/' line 

239 lend = " /\n" 

240 else: 

241 # Ends with an $END or &END 

242 lend = " %sEND\n" % gchar 

243 # Insert the lines in reverse order 

244 self.lines.insert(ibeg, lend) 

245 self.lines.insert(ibeg, " %s%s\n" % (gchar, grp)) 

246 # Update the namelist info 

247 self.UpdateNamelist()

249 # Find a list by name (and index if repeated) 

250 def GetGroupByName(self, grp, igrp=0): 

251 r"""Get index of group with a specific name 

252 

253 :Call: 

254 >>> i = nml.GetGroupByName(grp, igrp=0) 

255 :Inputs: 

256 *nml*: :class:`cape.name.ist2.Namelist2` 

257 Interface to namelist with repeated lists 

258 *grp*: :class:`str` 

259 Name of namelist group 

260 *igrp*: {``0``} | :class:`int` 

261 If multiple matches, return match number *igrp* 

262 :Outputs: 

263 *i*: :class:`int` | :class:`np.ndarray`\ [:class:`int`] 

264 Group index of requested match 

265 :Versions: 

266 * 2016-01-31 ``@ddalle``: Version 1.0 

267 """ 

268 # Search based on lower-case names 

269 grps = np.array([gi.lower() for gi in self.Groups]) 

270 # Find the all indices that match 

271 I0 = np.where(grps == grp.lower())[0] 

272 # Process output 

273 if igrp is None: 

274 # Return all matches 

275 return I0 

276 elif len(I0) == 0: 

277 # No match 

278 return KeyError( 

279 "Namelist '%s' has no list '%s'" 

280 % (self.fname, grp)) 

281 elif len(I0) < igrp: 

282 # Not enough matches 

283 return ValueError( 

284 "Namelist '%s' has fewer than %i lists named '%s'" 

285 % (self.fname, igrp, grp)) 

286 else: 

287 # Return the requested match 

288 return I0[igrp]

290 # Turn a namelist into a dict 

291 def ReadGroupIndex(self, igrp): 

292 r"""Read group *igrp* and return a dictionary 

293 

294 The output is a :class:`dict` such as the following 

295 

296 ``{'FSMACH': '0.8', 'ALPHA': '2.0'}`` 

297 

298 If a parameter has an index specification, such as ``"PAR(2) = 1.0"``, 

299 the dictionary will have the following format for such keys. 

300 

301 ``{'PAR': {2: 1.0}}`` 

302 

303 :Call: 

304 >>> d = nml.ReadGroupIndex(igrp) 

305 :Inputs: 

306 *nml*: :class:`cape.filecntl.namelist2.Namelist2` 

307 Interface to namelist with repeated lists 

308 *igrp*: :class:`int` 

309 Group index to read 

310 :Outputs: 

311 *d*: :class:`dict`\ [:class:`str`] 

312 Raw (uncoverted) values of the dict 

313 :Versions: 

314 * 2016-01-29 ``@ddalle``: Version 1.0 

315 """ 

316 # Initialize the dictionary 

317 d = {} 

318 # Get index of starting line 

319 ibeg = self.ibeg[igrp] 

320 # Get index of end line 

321 if igrp == len(self.ibeg): 

322 # Use the last line 

323 iend = len(self.lines)+1 

324 else: 

325 # Use the line before the start of the next line 

326 iend = self.ibeg[igrp+1] 

327 # Get the lines 

328 lines = [line.strip() for line in self.lines[ibeg:iend]] 

329 # Process the first line to catch keys in the opening line 

330 vals = lines[0].split() 

331 # Check for multiple entries 

332 if len(vals) > 1: 

333 # Read the additional values 

334 line = ' '.join(vals[1:]) 

335 # Process it 

336 di = self.ReadKeysFromLine(line) 

337 # Append the keys. 

338 for k in di: d[k] = di[k] 

339 # Loop through the lines. 

340 for line in lines[1:]: 

341 # Process the line 

342 di = self.ReadKeysFromLine(line) 

343 # Check for end 

344 if di == -1: break 

345 # Append the keys. 

346 for k in di: d[k] = di[k] 

347 # Output 

348 return d

457 # Try to read a key from a line 

458 def GetKeyFromLine(self, line, key, i=None): 

459 """Read the value of a key from a line 

460 

461 :Call: 

462 >>> q, val = nml.GetKeyFromLine(line, key, i=None) 

463 :Inputs: 

464 *nml*: :class:`cape.filecntl.namelist2.Namelist2` 

465 Interface to namelist with repeated lists 

466 *line*: :class:`str` 

467 A line of text that may or may not contain the value of *key* 

468 *key*: :class:`str` 

469 Name of key 

470 *i*: {``None``} | ``":"`` | :class:`int` 

471 Index to use in the namelist, e.g. "BCPAR(*i*)" 

472 :Outputs: 

473 *q*: :class:`bool` 

474 Whether or not the key was found in the line 

475 *val*: :class:`str` | :class:`float` | :class:`int` | :class:`bool` 

476 Value of the key, if found 

477 :Versions: 

478 * 2016-01-29 ``@ddalle``: Version 1.0 

479 * 2016-01-30 ``@ddalle``: Case-insensitive 

480 * 2016-08-29 ``@ddalle``: Added index capability 

481 """ 

482 # Check for the line 

483 if key.lower() not in line.lower(): 

484 # Key not read in this text 

485 return False, None 

486 # Initialize text remaining. 

487 tend = line 

488 while tend != "": 

489 # Read the first key in the remaining text. 

490 try: 

491 tend, ki, vi, ii = self.PopLine(tend) 

492 except Exception as e: 

493 raise ValueError( 

494 ("Failure reading line or partial line:\n") + 

495 (" '%s'\n" % tend) + 

496 ("Original error:\n") + 

497 (" '%s'" % e.message)) 

498 # Check for a match. 

499 if ki.lower() == key.lower() and ii == i: 

500 # Use the value from this key. 

501 return True, self.ConvertToVal(vi) 

502 # If this point is reached, the key name is hiding in a comment or str 

503 return False, None

534 def SetKeyInGroupIndex(self, igrp, key, val, i=None): 

535 r"""Set the value of a key in a group by index 

536 

537 If the key is not set in the present text, add it as a new line. 

538 The contents of the file control's text (in *nml.lines*) will be 

539 edited, and the list indices will be updated if a line is added. 

540 

541 :Call: 

542 >>> nml.SetKeyInGroupIndex(igrp, key, val, i=None) 

543 :Inputs: 

544 *nml*: :class:`cape.filecntl.namelist2.Namelist2` 

545 File control instance for old-style Fortran namelist 

546 *igrp*: :class:`int` 

547 Index of namelist to edit 

548 *key*: :class:`str` 

549 Name of key to alter or set 

550 *val*: :class:`any` 

551 Value to use for *key* 

552 *i*: {``None``} | ``":"`` | :class:`int` 

553 Index to use in the namelist, e.g. "BCPAR(*i*)" 

554 :Versions: 

555 * 2015-01-30 ``@ddalle``: Version 1.0 

556 * 2016-08-29 ``@ddalle``: Added index capability 

557 """ 

558 # Get index of starting and end lines 

559 ibeg = self.ibeg[igrp] 

560 iend = self.iend[igrp] 

561 # Initialize the boolean indicator of a match in existing text 

562 q = False 

563 # Loop through the lines 

564 for j in range(ibeg, iend): 

565 # Get the line. 

566 line = self.lines[j] 

567 # Try to set the key in this line 

568 q, line = self.SetKeyInLine(line, key, val, i=i) 

569 # Check for match. 

570 if q: 

571 # Set this line in the FC's text and exit 

572 self.lines[j] = line 

573 return 

574 # If no match found, nothing to delete 

575 if val is None: 

576 return 

577 # If no match found in existing text, add a line. 

578 if i is None: 

579 # No index 

580 line = ' %s = %s,\n' % (key, self.ConvertToText(val)) 

581 else: 

582 # Specify line with an index 

583 line = ' %s(%s) = %s,\n' % (key, i, self.ConvertToText(val)) 

584 # Insert the line. 

585 self.lines = self.lines[:iend] + [line] + self.lines[iend:] 

586 # Update the namelist indices. 

587 self.ibeg[igrp+1:] += 1 

588 self.iend[igrp:] += 1

616 # Check if the key is present in the line of the text. 

617 if key not in line: 

618 return False, line 

619 # Initialize prior and remaining text 

620 tbeg = "" 

621 tend = line 

622 # Loop through keys in this line 

623 while True: 

624 # Read the first key in the remaining line. 

625 try: 

626 txt, ki, vi, ii = self.PopLine(tend) 

627 except Exception as e: 

628 raise ValueError( 

629 ("Failure reading line or partial line:\n") + 

630 (" '%s'\n" % tend) + 

631 ("Original error:\n") + 

632 (" '%s'" % e.message)) 

633 # Check if the line is empty. 

634 if ki is None: 

635 # No match in this line. 

636 return False, line 

637 # Check if the key matches the target. 

638 if ki.lower() == key.lower() and ii == i: 

639 # Match found; exit and remember remaining text 

640 tbeg += tend[:tend.index(ki)] 

641 tend = txt 

642 break 

643 # Check if the line is empty. 

644 if txt == "": 

645 # No match in this line. 

646 return False, line 

647 # Otherwise, append to the prefix text and keep looking. 

648 tbeg += tend[:tend.index(txt)] 

649 # Update the text remaining 

650 tend = txt 

651 # If the value is ``None``, delete the entry. 

652 if val is None: 

653 # Use the beginning and remaining text Only 

654 line = "%s%s\n" % (tbeg, tend.lstrip()) 

655 else: 

656 # Convert value to text 

657 sval = self.ConvertToText(val) 

658 # Check for index 

659 if i is None: 

660 # No index 

661 line = "%s%s = %s,%s\n" % (tbeg, key, sval, tend) 

662 else: 

663 # Set an index as well 

664 line = "%s%s(%s) = %s,%s\n" % (tbeg, key, i, sval, tend) 

665 return True, line

667 # Pop line 

668 def PopLine(self, line): 

669 r"""Read the left-most key from line and return rest of line 

670 

671 :Call: 

672 >>> txt, key, val, i = nml.PopLine(line) 

673 :Inputs: 

674 *nml*: :class:`cape.filecntl.namelist2.Namelist2` 

675 Interface to namelist with repeated lists 

676 *line*: :class:`str` 

677 One line of namelist text 

678 :Outputs: 

679 *txt*: :class:`str` 

680 Remaining text in *line* after first key has been read 

681 *key*: :class:`str` 

682 Name of first key read from *line* 

683 *val*: ``None`` | :class:`str` 

684 Raw (unconverted) value of *key* 

685 *i*: {``None``} | ``':"`` | :class:`int` 

686 Vector index if specified 

687 :Versions: 

688 * 2016-01-29 ``@ddalle``: Version 1.0 

689 * 2016-08-29 ``@ddalle``: Version 1.1 

690 - Add indices, e.g. BCPAR(2) 

691 """ 

692 # Strip line 

693 txt = line.strip() 

694 # Check for comment 

695 if txt.startswith('!'): 

696 return '', None, None, None 

697 # Check for start of namelist 

698 if txt and txt[0] in ["$", "&"]: 

699 # Get the stuff. 

700 V = line.split() 

701 # Check for remaining stuff to process. 

702 if len(V) > 1: 

703 # Process the rest of the line 

704 txt = ' '.join(V[1:]) 

705 else: 

706 # Nothing else in the line 

707 txt = '' 

708 # Check for empty key 

709 if txt == "": 

710 return txt, None, None, None 

711 # Split on the equals signs 

712 vals = txt.split("=") 

713 # Remaining text 

714 txt = '='.join(vals[1:]) 

715 # Get the name of the key. 

716 key = vals[0].strip() 

717 # Attempt to split for index 

718 K = key.split('(') 

719 # Check for successful split 

720 if len(K) > 1: 

721 # Split by key and index 

722 key = K[0] 

723 # Template: "PAR(3) =" or "PAR(:) =" 

724 i = K[1].rstrip(')') 

725 # Check for ':' 

726 if i != ':': i = int(i) 

727 else: 

728 # No index 

729 i = None 

730 # Deal with quotes or no quotes 

731 if len(vals) == 1: 

732 # No value, last key in the line 

733 txt = '' 

734 val = None 

735 elif len(vals) == 2: 

736 # Last value in the line 

737 txt = '' 

738 val = vals[1].rstrip(',').strip() 

739 # Check for trivial value 

740 if val == "": val = None 

741 elif txt.startswith('"'): 

742 # Check for a second quote 

743 if '"' not in txt[1:]: 

744 # Unterminated string 

745 raise ValueError( 

746 "Namelist line '%s' could not be interpreted" % line) 

747 # Find it 

748 iq = txt[1:].index('"') + 1 

749 # Split of at this point 

750 val = txt[1:iq] 

751 # Remaining text (?) 

752 if len(txt) > iq+1: 

753 txt = txt[iq+1:] 

754 else: 

755 txt = '' 

756 elif txt.startswith("'"): 

757 # Check for a second quote 

758 if "'" not in txt[1:]: 

759 # Unterminated string 

760 raise ValueError( 

761 "Namelist line '%s' could not be interpreted" % line) 

762 # Find it 

763 iq = txt[1:].index("'") + 1 

764 # Split of at this point 

765 val = txt[1:iq] 

766 # Remaining text (?) 

767 if len(txt) > iq+1: 

768 txt = txt[iq+1:] 

769 else: 

770 txt = '' 

771 else: 

772 # Read until just before the next '=' 

773 subvals = vals[1].split(',') 

774 # Rejoin but without name of next key 

775 val = ','.join(subvals[:-1]) 

776 # Remaining text 

777 txt = subvals[-1] + '=' + '='.join(vals[2:]) 

778 # Ouptut 

779 return txt, key, val, i

781 # Conversion 

782 def ConvertToVal(self, val): 

783 r"""Convert text to Python based on a series of rules 

784 

785 :Call: 

786 >>> v = nml.ConvertToVal(val) 

787 :Inputs: 

788 *nml*: :class:`cape.filecntl.namelist.Namelist` 

789 File control instance for :file:`fun3d.nml` 

790 *val*: :class:`str` | :class:`unicode` 

791 Text of the value from file 

792 :Outputs: 

793 *v*: ``str`` | ``int`` | ``float`` | ``bool`` | ``list`` 

794 Evaluated value of the text 

795 :Versions: 

796 * 2015-10-16 ``@ddalle``: Version 1.0 

797 * 2016-01-29 ``@ddalle``: Added boolean shortcuts, ``.T.`` 

798 """ 

799 # Check inputs. 

800 if type(val).__name__ not in ['str', 'unicode']: 

801 # Not a string; return as is. 

802 return val 

803 # Split to parts 

804 V = val.split(',') 

805 # Check the value. 

806 try: 

807 # Check the value. 

808 if ('"' in val) or ("'" in val): 

809 # It's a string. Remove the quotes. 

810 return eval(val) 

811 elif val.lower() in [".false.", ".f."]: 

812 # Boolean 

813 return False 

814 elif val.lower() in [".true.", ".t."]: 

815 # Boolean 

816 return True 

817 elif len(V) == 0: 

818 # Nothing here. 

819 return None 

820 elif len(V) == 1: 

821 # Convert to float/integer 

822 return eval(val) 

823 else: 

824 # List 

825 return [eval(v) for v in V] 

826 except Exception: 

827 # Give it back, whatever it was. 

828 return val

830 # Conversion to text 

831 def ConvertToText(self, v): 

832 """Convert a scalar value to text to write in the namelist file 

833 

834 :Call: 

835 >>> val = nml.ConvertToText(v) 

836 :Inputs: 

837 *nml*: :class:`cape.filecntl.namelist2.Namelist2` 

838 File control instance for old-style Fortran namelist 

839 *v*: :class:`str` | :class:`int` | :class:`float` | :class:`list` 

840 Evaluated value of the text 

841 :Outputs: 

842 *val*: :class:`str` | :class:`unicode` 

843 Text of the value from file 

844 :Versions: 

845 * 2015-10-16 ``@ddalle``: Version 1.0 

846 """ 

847 # Get the type 

848 t = type(v).__name__ 

849 # Form the output line. 

850 if t in ['str', 'unicode']: 

851 # Force quotes 

852 return "'%s'" % v 

853 elif t in ['bool'] and v: 

854 # Boolean 

855 return ".T." 

856 elif t in ['bool']: 

857 # Boolean 

858 return ".F." 

859 elif type(v).__name__ in ['list', 'ndarray']: 

860 # List (convert to string first) 

861 V = [str(vi) for vi in v] 

862 return ", ".join(V) 

863 else: 

864 # Use the built-in string converter 

865 return str(v)
