1""" 

2:mod:`cape.pyover.lineLoad`: Sectional loads module 

3===================================================== 

4 

5This module contains functions for reading and processing sectional loads. It 

6is a version of :mod:`cape.cfdx.lineLoad` that is closely tied to 

7:mod:`cape.pyover.dataBook`. 

8 

9It provides the primary class :class:`DBLineLoad`, which 

10is a subclass of :class:`cape.cfdx.dataBook.DBBase`. This class is an interface to 

11all line load data for a specific surface component. 

12 

13For reading the sectional load for a single solution on one component (which 

14includes multiple coefficients), there is another class :class:`CaseLL`. 

15 

16Finally, reading seam curves from individual cases utilizes the class 

17:class:`CaseSeam`. 

18 

19:See also: 

20 * :mod:`cape.cfdx.lineLoad` 

21 * :mod:`cape.cfdx.dataBook` 

22 * :mod:`cape.pyover.dataBook` 

23"""

420 # Get file 

421 def GetTriqFile(self): 

422 """Get most recent ``triq`` file and its associated iterations 

423  

424 :Call: 

425 >>> qpre, fq, n, i0, i1 = DBL.GetTriqFile() 

426 :Inputs: 

427 *DBL*: :class:`pyCart.lineLoad.DBLineLoad` 

428 Instance of line load data book 

429 :Outputs: 

430 *qpre*: {``False``} 

431 Whether or not to convert file from other format 

432 *fq*: :class:`str` 

433 Name of ``triq`` file 

434 *n*: :class:`int` 

435 Number of iterations included 

436 *i0*: :class:`int` 

437 First iteration in the averaging 

438 *i1*: :class:`int` 

439 Last iteration in the averaging 

440 :Versions: 

441 * 2016-12-19 ``@ddalle``: Added to the module 

442 """ 

443 # Get properties of triq file 

444 fq, n, i0, i1 = case.GetQFile(self.fqi) 

445 # Get the corresponding .triq file name 

446 ftriq = os.path.join('lineload', 'grid.i.triq') 

447 # Check for 'q.strt' 

448 if os.path.isfile(fq): 

449 # Source file exists 

450 fsrc = os.path.realpath(fq) 

451 else: 

452 # No source just yet 

453 fsrc = None 

454 # Check if the TRIQ file exists 

455 if os.path.isfile(ftriq) and fsrc and os.path.isfile(fsrc): 

456 # Check modification dates 

457 if os.path.getmtime(ftriq) < os.path.getmtime(fsrc): 

458 # 'grid.i.triq' exists, but Q file is newer 

459 qpre = True 

460 else: 

461 # Triq file exists and is up-to-date 

462 qpre = False 

463 else: 

464 # Need to run ``overint`` to get triq file 

465 qpre = True 

466 # If "grid.i.triq" file is up-to-date, check if it has the component 

467 if not qpre: 

468 # Read the existing "mixsur.i" file 

469 fmixsur = os.path.join("lineload", "mixsur.i") 

470 # Open the file 

471 if os.path.isfile(fmixsur): 

472 # Read that file 

473 cfg = config.ConfigMIXSUR(fmixsur) 

474 compID = self.opts.get_DataBookCompID(self.comp) 

475 # Check if the component is present 

476 if compID not in cfg.faces: 

477 # The "grid.i.triq" does not include the component we need 

478 qpre = True 

479 else: 

480 # No knowledge of components; must run overint  

481 qpre = True 

482 # Output 

483 return qpre, fq, n, i0, i1
